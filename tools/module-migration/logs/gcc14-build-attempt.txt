- std::list::insert_range - C++23 feature, NOT in libstdc++ 14.2

These features are from P1206R7 (ranges::to, container insert_range) which is
still being implemented across STL vendors.

AVAILABLE STL IMPLEMENTATIONS:
1. libstdc++ (GCC 14.2.0) - INCOMPLETE C++23 support
